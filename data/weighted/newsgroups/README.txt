Setup:

Install sklearn using
pip3 install sklearn
or
pip3 install --user sklearn

Generating the instances:

run python3 generate.py
5 instances will be generated each roughly 70MB large


Based on:
https://ojs.aaai.org/index.php/AAAI/article/view/4450
https://scikit-learn.org/0.19/datasets/twenty_newsgroups.html

20 newsgroups also used in
https://www.aclweb.org/anthology/W09-1803/
but processing of the data is too archaic to reproduce
